# 🍞 Dulce Herencia - ERP para Panadería

Sistema de gestión integral para panaderías desarrollado con JavaScript vanilla y Supabase como backend.

## 🚀 Características

- ✅ **Punto de Venta**: Registro de ventas en tiempo real
- ✅ **Catálogo de Productos**: Gestión de productos con presentaciones
- ✅ **Historial de Ventas**: Consulta y análisis de ventas
- ✅ **Gestión de Empleados**: Control de personal y cálculo de planilla
- ✅ **Inventario de Materia Prima**: Control de stock e insumos
- ✅ **Instrumentos**: Registro de inversiones en equipamiento
- ✅ **Análisis de Ganancias**: Métricas, metas y rentabilidad
- ✅ **Gráficas y Estadísticas**: Visualización de datos

## 🛠️ Tecnologías

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: [Supabase](https://supabase.com/) (PostgreSQL)
- **Despliegue**: [Vercel](https://vercel.com/)
